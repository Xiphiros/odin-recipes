# Odin Recipes (Inazuman Cuisine: A Collection of Unique Recipes)

## Overview

The purpose of this repository is to make a simple recipes website with the basic knowledge of HTML5 taught in the Foundations course so far, using the basic HTML boilerplate and tags (and their respective attributes, too) for images, links, lists, and different tags to display text with the most basic tools available.

## Project Description

- **Project:** Odin Recipes
- **Course:** The Odin Project - Foundations
- **Lesson:** [Project: Recipes](https://www.theodinproject.com/lessons/foundations-recipes), [The Cascade](https://www.theodinproject.com/lessons/foundations-the-cascade), [Block and Inline](https://www.theodinproject.com/lessons/foundations-block-and-inline)

## Features

- A homepage, using the elements taught in the curriculum so far (the basic HTML boilerplate, headers, lists, images and links to each recipe using relative links).
- Multiple recipe pages, each with images and steps to make their specific dish, using unordered and ordered lists, strong tags, and absolute links to the Genshin Wiki for in-game information of the dish.
- Basic styling using both HTML5 and CSS through a external stylesheet.

## Built With

- **HTML5**
- **CSS**

## Setup and Installation

Clone the repository and open index.html, or preferably open the [Github Pages site](https://xiphiros.github.io/odin-recipes/).
